Bienvenue sur l'application Esup-Signature.


  • Pour effectuer des signatures calligraphiques sur un PDF il faut au préalable déposer un fichier image de votre signature dans vos paramètres : Mes paramètres
    Un tutoriel video est disponible à cette adresse : Insérer sa signature
  • Cliquez ici pour en savoir plus sur les autres types de signature
    • Pour utiliser un certificat électronique pour signer, vous pouvez importer un certificat au format PKCS12 dans votre profil.
    • Enfin, il est possible de signer électroniquement des documents à l'aide d'un support cryptographique USB (agréé eIDAS, ex RGS **).
      Pour cela vous devez installer l'application NexU disponible ici : NexU
      Cette application se lancera automatiquement lors du processus de "signature avec clé matérielle"

Un guide utilisateur est accessible sur le portail de la communauté ESUP : Documentation utilisateur
