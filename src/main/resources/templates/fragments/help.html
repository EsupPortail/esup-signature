Bienvenue sur l'application Esup-Signature.
  • Pour effectuer des signatures calligraphiques sur un PDF il faut déposer un fichier image de votre signature dans vos paramètres : Mes paramètres
    Un tutoriel video est disponible à cette adresse : Insérer sa signature
  • Pour utiliser la signature électronique non qualifiée, vous pouvez importer un certificat au format PKCS12 dans votre profil.
  • Enfin il est possible d'utiliser un support cryptographique (certificat RGS**). Pour cela vous devez installer l'application NexU disponible ici : NexU
    Cette application se lancera automatiquement lors du processus de "signature avec clé matérielle"

Vous pouvez modifier / compléter vos paramètres à tous moments en utilisant le bouton paramètres en haut à droite.

Un guide utilisateur est accessible sur le portail de la communauté ESUP : Documentation utilisateur
