ProConnect - Authentification professionnelle

ProConnect est le moyen d'authentification officiel de l'État pour les professionnels. Il remplace AgentConnect et permet aux professionnels de se connecter de manière simple et sécurisée aux services numériques de l'État.
Qui peut utiliser ProConnect ?
  • Agents publics (fonction publique d'État, territoriale et hospitalière)
  • Professionnels du secteur privé disposant d'un numéro SIRET
  • Consultants indépendants et travailleurs rattachés à une organisation
Comment se connecter avec ProConnect ?

1 Cliquez sur le bouton "S'identifier avec ProConnect"

2 Saisissez votre adresse email professionnelle

3 ProConnect vous redirige automatiquement vers votre fournisseur d'identité (Cerbère, Passage2, etc.)

4 Identifiez-vous avec vos identifiants habituels

5 Vous êtes redirigé vers Esup-Signature pour signer votre document

Important : L'adresse email utilisée avec ProConnect doit être identique à celle qui a reçu le lien de signature.

FranceConnect - Authentification pour les particuliers

FranceConnect est la solution proposée par l'État pour simplifier la connexion des particuliers aux services publics en ligne. Plus de 1400 services utilisent FranceConnect pour sécuriser et faciliter vos démarches.
Qui peut utiliser FranceConnect ?
  • Tous les particuliers disposant d'un compte chez un des fournisseurs d'identité
  • Citoyens français et résidents en France
  • Personnes majeures (accessible dès 16 ans)
Comment se connecter avec FranceConnect ?

1 Cliquez sur le bouton "S'identifier avec FranceConnect"

2 Choisissez votre fournisseur d'identité parmi :

  • impots.gouv.fr (recommandé)
  • Ameli (Assurance Maladie)
  • L'Identité Numérique La Poste
  • MSA (Mutualité Sociale Agricole)
  • France Identité
  • Mobile Connect et Moi
  • Yris ou TrustMe

3 Saisissez vos identifiants sur le site du fournisseur choisi

4 Vous êtes redirigé vers Esup-Signature pour signer votre document

Important : L'adresse email associée à votre compte FranceConnect doit être identique à celle qui a reçu le lien de signature.

Authentification par code SMS (OTP)

Cette méthode est proposée en alternative. Nous vous recommandons de privilégier ProConnect ou FranceConnect pour une authentification plus sécurisée.
Comment utiliser le code SMS ?

1 Cliquez sur "Autre méthode" pour déplier les options alternatives

2 Saisissez votre numéro de téléphone mobile avec l'indicatif pays

3 Cliquez sur "Envoyer"

4 Vous recevez un code à 6 chiffres par SMS

5 Saisissez ce code dans le champ prévu et cliquez sur "Confirmer"

Le code SMS est valable pendant une durée limitée. En cas d'expiration, vous pouvez demander un nouveau code.

Problèmes fréquents et solutions

Je ne reçois pas le code SMS
  • Vérifiez que votre numéro de téléphone est correct
  • Vérifiez l'indicatif pays (par défaut +33 pour la France)
  • Attendez quelques minutes, les SMS peuvent parfois être retardés
  • Vérifiez que vous avez du réseau mobile
  • Essayez d'utiliser ProConnect ou FranceConnect à la place
ProConnect ne reconnaît pas mon adresse email
  • Vérifiez que vous utilisez bien votre adresse email professionnelle
  • L'adresse doit être identique à celle qui a reçu l'invitation
  • Contactez l'administrateur de votre organisation si nécessaire
FranceConnect : problème de connexion
  • Vérifiez vos identifiants sur le site du fournisseur (impots.gouv.fr, ameli.fr, etc.)
  • Assurez-vous d'utiliser l'adresse email qui a reçu l'invitation
  • Si vous avez oublié vos identifiants, utilisez la fonction "Mot de passe oublié" sur le site du fournisseur
  • Vérifiez qu'il n'y a pas de maintenance en cours sur FranceConnect
Je n'arrive pas à me connecter
  • Assurez-vous que votre navigateur est à jour
  • Désactivez temporairement les bloqueurs de publicités
  • Essayez avec un autre navigateur (Chrome, Firefox, Edge)
  • Videz le cache de votre navigateur
Quelle méthode choisir : ProConnect ou FranceConnect ?
  • Si vous êtes un professionnel (agent public ou secteur privé) → utilisez ProConnect
  • Si vous êtes un particulier → utilisez FranceConnect
  • En cas de doute, essayez les deux méthodes : elles sont toutes deux sécurisées

Sécurité et confidentialité

Vos données sont protégées
  • ProConnect et FranceConnect utilisent les standards de sécurité les plus élevés (protocole OpenID Connect)
  • Vos données d'authentification sont hébergées en France
  • Aucune donnée personnelle n'est partagée sans votre consentement
  • La connexion est chiffrée de bout en bout
  • Ces services sont officiels et gratuits
