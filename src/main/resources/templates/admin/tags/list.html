Les groupes de tags définissent les colonnes affichées dans la liste des demandes de signature. Chaque groupe possède une couleur. Les tags d'un groupe sont affichés comme badges colorés dans la colonne correspondante.
Via l'API : paramètre tags=NomDuGroupe/NomDuTag — les tags inexistants sont créés automatiquement.
