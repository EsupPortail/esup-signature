Bienvenue sur l'application Esup-Signature.
  • Pour effectuer des signatures calligraphiques sur un PDF il faut déposer un fichier image de votre signature dans vos paramètres : Mes paramètres
  • Pour utiliser la signature électronique, il est possible d'importer un keystore au format PKCS12 dans votre profil.
  • Enfin il est possible d'utiliser une clé matérielle. Pour cela vous devez installer l'application NexU disponible ici : NexU

Vous pouvez modifier / compléter vos paramètres à tous moments en utilisant le bouton paramètres en haut à droite.
